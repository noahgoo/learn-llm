Self-attention is permutation-equivariant: shuffle the inputs and the
outputs shuffle identically. Position must therefore be injected
explicitly. The input to block 0 is **x<sub>i</sub> = W<sub>E</sub>[t<sub>i</sub>] +
W<sub>P</sub>[i]** — token embedding plus position embedding, elementwise
sum in ℝ<sup>768</sup>.

The original Transformer used fixed **sinusoidal** encodings —
sin/cos at geometrically spaced frequencies, chosen so any offset is a
linear function of the encoding, with no learned parameters.<Cite id="vaswani2017" />
GPT-2 instead **learns** W<sub>P</sub> ∈ ℝ<sup>1024×768</sup> as ordinary
parameters — simpler, works as well, but caps context at 1024 positions:
there is no row to look up for position 1025.<Cite id="radford2019" />

random: training organized neighboring positions into smooth
trajectories.
