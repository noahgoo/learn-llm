Self-attention is permutation-equivariant: shuffle the inputs and the
outputs shuffle identically. Position must be injected explicitly.
Three generations of solutions:

- **Sinusoidal** (original Transformer): fixed sin/cos at geometrically
  spaced frequencies; no parameters; any offset is a linear function of
  the encoding.<Cite id="vaswani2017" />
- **Learned absolute** (GPT-2, the model here): W<sub>P</sub> ∈
  ℝ<sup>1024×768</sup> trained like any weight; block 0's input is
  x<sub>i</sub> = W<sub>E</sub>[t<sub>i</sub>] + W<sub>P</sub>[i]. Simple,
  but context is capped — there is no row for position 1025.<Cite id="radford2019" />
- **Rotary (RoPE)** (most modern LLMs): rotate each Q/K pair by a
  position-dependent angle inside attention itself, so dot products
  depend on *relative* offsets — no addition into the stream, better
  length behavior.<Cite id="su2021" />
